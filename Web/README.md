# 知行录

> 一个基于 Nuxt 4 + Nuxt UI 构建的个人知识管理 Web 应用。

## 技术栈

| 类别 | 技术 |
| --- | --- |
| 框架 | Nuxt 4 + Vue 3 |
| UI 组件库 | Nuxt UI + TailwindCSS |
| 状态管理 | Pinia |
| Markdown 编辑器 | TipTap (Vue 3) |
| Markdown 渲染 | @nuxtjs/mdc |
| HTTP 客户端 | Axios |
| 工具库 | @vueuse/core |

## 开发

安装依赖：

```bash
pnpm install
```

启动开发服务器（`http://localhost:3000`）：

```bash
pnpm dev
```

## 构建

```bash
pnpm build
```
